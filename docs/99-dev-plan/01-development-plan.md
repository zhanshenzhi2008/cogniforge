## 测试技术方案

### 后端测试 (Go)

| 测试类型 | 工具 | 说明 |
|---------|------|------|
| 单元测试 | Go testing + testify | 业务逻辑测试 |
| HTTP 测试 | httptest | API 端点测试 |
| Mock | testify/mock | 依赖模拟 |

**测试命令**:
```bash
cd gateway
go test ./... -v -cover          # 运行所有测试
go test ./... -coverprofile=coverage.out  # 生成覆盖率报告
```

### 前端测试 (Vue/Nuxt)

| 测试类型 | 工具 | 说明 |
|---------|------|------|
| 单元测试 | Vitest | 组件和 Composables 测试 |
| 组件测试 | @vue/test-utils | Vue 组件测试 |
| E2E 测试 | Playwright | 端到端测试 |

**测试命令**:
```bash
# 前端
pnpm test              # 运行单元测试
pnpm test:watch        # 监听模式
pnpm test:e2e          # E2E 测试
```

### 测试覆盖率目标

| 模块 | 最低覆盖率 |
|------|-----------|
| 后端 Handler | 70% |
| 后端 Service | 80% |
| 前端 Composables | 80% |
| 前端组件 | 60% |
